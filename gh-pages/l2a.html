Method to convert age to length

Arguments

object
an FLQuant, or FLStockLen object.
model
a a4aGr object
stat
the aggregation statistic, must be \"mean\" or \"sum\". Only used if object is an FLQuant.
plusgroup
the plusgroup of the stock. Only used if the object is a FLStockLen.

Value

an age based FLQuant, FLStock

Description

Method to convert age to length

Examples

# red fish
# M=0.05; Linf=58.5, k=0.086
mm <- matrix(NA, ncol=3, nrow=3)
diag(mm) <- c(50, 0.001,0.001)
mm[upper.tri(mm)] <- mm[lower.tri(mm)] <- c(0.1,0.01,0.00004)
vbObj <- a4aGr(grMod=~linf*(1-exp(-k*(t-t0))), grInvMod=~t0-1/k*log(1-len/linf), params=FLPar(linf=58.5, k=0.086, t0=0.001, units=c("cm","ano-1","ano")), vcov=mm, distr="norm")
data(rfLen)
cth <- catch.n(rfLen.stk)
# both with iter=1
cthA1 <- l2a(cth, vbObj)
