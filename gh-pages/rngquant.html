quant range extract and replacement

Usage

rngquant(object, ...)

Arguments

object
an FLComp object
value
a vector with max and min quant range to replace the object info

Value

a vector object when extracting, or an FLComp object when replacing

Description

Methods to extract from FLComp objects the quant range, or to replace its value.

Examples

data(ple4.index)
data(rfLen)
rngquant(ple4.index)
