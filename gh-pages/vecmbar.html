M age vector

Usage

vecmbar(object, ...)

Arguments

object
a a4aM object

Value

an vector object

Description

Method to extract from a4aM objects the vector of ages for the mean M.

Examples

mod1 <- FLModelSim(model=~exp(-age-0.5))
mod2 <- FLModelSim(model=~1.5*k, params=FLPar(k=0.4))
m1 <- a4aM(shape=mod1, level=mod2)
rngmbar(m1)<-c(1,5)
vecmbar(m1)
